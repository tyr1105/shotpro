## ✨ What it does

ShotPro takes your plain screenshots and wraps them in beautiful backgrounds with rounded corners, shadows, and optional watermarks — perfect for sharing on Twitter/X, LinkedIn, blog posts, or documentation.

## 🎯 Features

- 🎨 **12 Gradient Presets** — Sunset, Deep Blue, Lavender, Aurora, and more
- 🖌️ **6 Solid Colors** — Clean white, warm gray, brand blue, dark mode ready
- 📐 **Custom Padding and Border Radius** — Fine-tune the spacing and roundness
- 🌑 **Dark Mode** — Full dark theme support
- 🔲 **Drop Shadows** — Light, medium, or heavy shadow intensity
- 💧 **Watermarks** — Custom text with adjustable opacity and size
- 📥 **Export as PNG** — High-quality 2x resolution output
- 📋 **Copy to Clipboard** — One-click paste into any app
- 🔒 **100% Private** — Everything runs in your browser. No server, no upload, no tracking

## 🛠️ Built With

- **React** + **Vite** — Fast development and build
- **Tailwind CSS** — Utility-first styling
- **html-to-image** — Client-side image generation
- Zero backend — 100% static site, hosted on GitHub Pages

## 🏃 Quick Start

```bash
git clone https://github.com/tyr1105/shotpro.git
cd shotpro
npm install
npm run dev
```
